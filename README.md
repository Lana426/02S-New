# 02S Customer Portal — Prototype

Interactive visioning prototype for McCarthy's **02S Customer Portal** — the project-team
workspace for planning, ordering, and managing rental equipment and services across a project.

Single self-contained `index.html`: no build step, no dependencies, no framework.
Deployed as a static site on Vercel.

## Local preview
Open `index.html` directly in any browser, or serve the folder:

    python3 -m http.server 8000

then visit http://localhost:8000

## Deploy (Vercel)
Static deploy — connect the repo to Vercel and deploy. No framework preset, build command,
or output directory configuration is required; Vercel serves `index.html` at the root.
